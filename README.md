# data-warehouse-ml
# 📊 Data Warehouse + Machine Learning

## 🧠 Sobre o projeto
Este projeto tem como objetivo a construção de uma solução completa de dados, envolvendo modelagem de Data Warehouse, implementação em PostgreSQL, análise exploratória de dados e aplicação de Machine Learning para geração de insights de negócio.

---

## 📌 Objetivos
- Construir um Data Warehouse em PostgreSQL
- Realizar ETL simplificado com dados reais
- Aplicar análise exploratória de dados (EDA)
- Implementar modelos de Machine Learning
- Gerar insights de negócio baseados em dados

---

## 🛠️ Tecnologias utilizadas
- Python
- Pandas
- Matplotlib
- Scikit-learn
- PostgreSQL
- Jupyter Notebook

---

## 🗄️ Data Warehouse
O Data Warehouse foi modelado utilizando o conceito de Star Schema, contendo:
- Tabela fato (vendas)
- Dimensões: cliente, produto, categoria, região e tempo

O banco foi implementado no PostgreSQL e exportado para versionamento no GitHub.

---

## 📊 Análise de Dados (EDA)
Foram realizadas análises exploratórias com foco em:
- Vendas por categoria
- Lucro por região
- Evolução temporal das vendas
- Produtos com maior desempenho

---

## 🤖 Machine Learning
Foram aplicados modelos de Machine Learning para análise e previsão de dados:

- KNN
- Decision Tree
- Random Forest
- Logistic Regression
- Regressão Linear

Os modelos foram avaliados utilizando métricas como:
- Acurácia
- Precision
- Recall
- F1-score
- Matriz de confusão

---

## 💡 Resultados e Conclusões
O projeto permitiu identificar padrões importantes nos dados e gerar previsões que auxiliam na tomada de decisão estratégica.
